# CloudCafe Product Overview

CloudCafe is a production-grade, multi-service coffee ordering platform demonstrating AWS best practices at scale. It simulates a Starbucks-scale operation (30K stores, 100M users, 5M daily orders).

## Core Purpose

- Demonstrate comprehensive AWS service integration (17 services)
- Showcase chaos engineering and resilience testing
- Provide production-ready patterns (auto-scaling, failover, caching)
- Enable observability through CloudWatch dashboards and metrics

## Key Features

- Polyglot microservices architecture (Python, Go, Node.js, Java)
- Built-in CPU stress scenarios with storytelling
- 12+ chaos engineering scripts for resilience testing
- Rich CloudWatch metrics and comprehensive dashboards
- Complete observability and monitoring

## Business Domain

Coffee ordering system with:
- Order management and processing
- Store inventory tracking
- Customer loyalty programs
- Menu catalog management
- Payment processing
- Real-time analytics

## Target Scale

- 30,000 stores
- 100 million users
- 5 million daily orders
- High availability and fault tolerance requirements
